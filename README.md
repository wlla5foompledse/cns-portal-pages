# cns-portal-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定网站或域名，而是作为通用前端页面资源的存储与分发集合。

## 目录说明

```
cns-portal-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

## 页面归档说明

- 所有页面均为独立、完整的 HTML 文件，可直接在浏览器中打开使用。
- 页面可能引用 `assets/` 目录下的资源文件，建议保持相对路径一致。
- 归档的页面不包含任何指向特定域名或外部服务的硬编码链接。

## 维护说明

- 欢迎提交包含新页面的 Pull Request，但请确保页面内容不涉及特定商业推广或诱导点击。
- 请保持页面结构的清晰与资源的合理组织。
- 如发现页面无法正常显示或资源缺失，可提交 Issue 反馈。
